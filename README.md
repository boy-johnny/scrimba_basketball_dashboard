# Basketball Dashboard

![preview](preview.png)

## Introduction

An interactive dashboard for basketball.

## Feature

- **countdown**
- **interactive counter**
- **start new period**

## Tech Stack

Use `html-css-javascript` to build frontend

## Getting Started

```bash
git clone https://github.com/boy-johnny/scrimba_basketball_dashboard/
npm install
npm start
```
